уже менее временный, но всё равно потом перепишется!

формат:
python ./main.py [command] [args]

Например:
python ./main.py help
python ./main.py init
python ./main.py add .


python ./main.py help [command] - выводит подсказку по команде (вроде бы пока там всё описано)

Команды:
init - первичная инициализация
add [маска для включаемых файлов] - добавление в индекс файлов, кроме тех, что указаны в .gitignore
commit [сообщение] - создание коммита (пока работает на создании копий файлов)
branch [название ветки] - создание новой ветки с того коммита, на котором находимся
checkout [название ветки] - переход на указанную ветку (пока формальный, состояние файлов не меняется)
help - тот самый help
